# Experiment 2: Simple To-Do List SPA

## Aim
To build a to-do list application with add, delete, and mark complete functionality.

## Procedure
1. Create form to add new tasks.
2. Use `useState` to manage task list.
3. Implement delete and complete toggle features.

## How to Run

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser to `http://localhost:5173`

## Features
- ✨ Add new tasks with input field
- ✨ Delete tasks individually
- ✨ Mark tasks as completed
- ✨ Visual feedback for completed tasks
- ✨ Enter key to add tasks
- ✨ Task counter
- ✨ Responsive design

## Technology Stack
- React 19.2.0
- Vite 7.2.4
- CSS3 with modern styling

## How It Works
- Type a task and press Enter or click "Add Task"
- Click the trash icon to delete a task
- Click a task to mark it as complete/incomplete
- Completed tasks show a strikethrough effect
