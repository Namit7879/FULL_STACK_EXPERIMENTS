# Experiment 1: Simple Counter SPA

## Aim
To build a basic counter application using React state management.

## Procedure
1. Create increment and decrement buttons.
2. Use `useState` to manage counter state.
3. Display current counter value.

## How to Run

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser to `http://localhost:5173`

## Features
- ✨ Increment counter with (+) button
- ✨ Decrement counter with (-) button
- ✨ Real-time counter value display
- ✨ Clean and intuitive UI
- ✨ Responsive design
- ✨ Smooth button hover effects

## Technology Stack
- React 19.2.0
- Vite 7.2.4
- CSS3 with modern styling
